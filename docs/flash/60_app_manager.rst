****************************
App Manager
****************************

.. note::

   This page is a **placeholder** — the App Manager ships in a future release.
   For now, use the `Stable Channel <10_stable_channel.html>`_ to flash firmware.

The App Manager will provide:

- **Bootloader flash via BOOTSEL** — flash the RP2350 co-processor from
  the browser using BOOTSEL mode (hold button on power-up).
- **System tools** — device diagnostics, partition info, hash verification.
- **Sideload** — install custom firmware builds directly from a local file
  without using a CDN channel.
