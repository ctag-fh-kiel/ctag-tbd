Welcome to the Sample Manager! Upload, organize, and assign audio samples to your TBD-16 kits. Drag & drop WAV files, build custom kits, and configure which samples load on each track at boot.

Sample Pool (Left Panel)

The Sample Pool shows all audio files on the TBD-16 SD card.

  1. Browse Folders — Click folder names to navigate. Use the breadcrumb path at the top to go back.
  2. Upload Samples — Click Upload or drag & drop WAV files onto the panel. Files upload directly to the current folder.
  3. Organize — Use New Folder to create subfolders. Use Select mode to move or delete multiple files at once.
  4. Preview — Click the ▶ button next to any sample to hear it through your browser.
Kit Editor (Right Panel)

The Kit Editor lets you assign samples to the 8 rompler tracks (banks of up to 32 slices each).

  1. Select a Kit — Use the dropdown at the top. Kits are stored in PSRAM and loaded at boot.
  2. Add Samples — Click + Add sample inside a bank, or drag files from the Sample Pool directly into a bank slot.
  3. Reorder — Drag sample rows within a bank to change slice order.
  4. Toggle View — Use the Banks toggle to switch between the compact bank list and expanded view.
  5. Save — Click Save to write the kit to SD card. Changes are not saved automatically.
Boot Default Presets

Use the Boot Defaults button (HDD icon in the header) to configure which machine, preset, and sample bank each track loads on power-up.

  1. Pick a Machine — For each of the 20 tracks, choose which synth engine to load.
  2. Pick a Preset — Select from the available sound presets for that machine.
  3. Rompler Tracks — For rompler-type tracks, also set the default Bank and Slice from your kit.
  4. Save to SD Card — Changes take effect on the next power cycle.
Import & Export Kits

Back up and share your kits using the export/import buttons in the Kit Editor header.

  • Export — Downloads the current kit as a JSON file. Does not include the audio files themselves — only the bank/slice assignments.
  • Import — Upload a previously exported kit JSON. The referenced samples must already exist on the SD card.
