Welcome to TBD-16! A 16-track drum machine & synth workstation. Use the channel strip at the top to pick a voice, then use the two sidebar modes to play or design sounds.

Presets (Performer Mode)

The PRESETS tab is for browsing, loading, and saving sounds — no DSP knowledge required.

  1. Select a Track — Click a channel in the strip at the top (CH 01–19). Each channel drives one synth voice.
  2. Filter by Macro — Use the MACRO dropdown to show only presets for a specific macro definition, or choose All Macros to see everything. A filter chip appears in the sidebar when filtered — click × to clear it.
  3. Load a Preset — Click any preset in the sidebar to load it instantly. The knobs update to show the preset's values. The MACRO dropdown syncs to show which macro the preset uses.
  4. Tweak Knobs — Drag the knobs in the center panel. Knobs with an orange MACRO label control multiple DSP parameters at once.
  5. Save Your Sound — Click Save As… to save your tweaked values as a new preset. Give it a name and category.
  6. Quick Actions — Randomize generates random values for inspiration. Init resets all knobs to defaults. Export/Import lets you share presets as JSON files.
Macros (Sound Designer Mode)

The MACROS tab is for creating and editing macro definitions — the mapping layer that turns raw DSP CC parameters into intuitive performer knobs.

Getting Started

  1. Select a Track — Click a channel tab. The MACHINE dropdown shows the synth engine on that track.
  2. Pick or Create — The sidebar lists existing macro definitions. Click one to edit, or click "+ Create New Definition".
  3. Name & ID — Set the definition name and ID in the header bar. The ID is auto-generated from the name for new definitions.
  4. Knob Preview — The default sub-tab shows a live preview of the knobs. Drag them to see how the mapped CC values change in real time.
  5. Macro Builder — Switch to this sub-tab to define parameter groups (pages of up to 4 knobs), set ranges, defaults, and output mappings.
  6. Save — Click Save in the header bar. Use Export/Import to share definitions as JSON.

Key Concepts

Output Mappings

Each output mapping row maps your virtual knob values to a DSP CC channel: finalValue = start + Σ(paramValue × mul ÷ div)

  • 1:1 Passthrough — One knob controls one CC directly. Use 1:1 Map All to auto-generate a full passthrough mapping as a starting point.
  • MACRO Knob — Add multiple sources to one CC output. A single "Punch" knob can simultaneously drive attack time, FM depth, and accent.
  • Constant Lock — Set only the start value (no sources). Locks a CC to a fixed value — great for parameters you don't want performers to change.
Sound Presets

Each macro definition can have sound presets — saved sets of knob values. Find them in the "Presets Using This Definition" section below the Macro Builder. Create, edit, and delete presets from there.

Signal Chain

When you move a knob, here's what happens:

  1. Knob Value (0–127) — the virtual parameter you control in the UI
  2. Mapping Formula — computes start + Σ(value × mul ÷ div), producing a CC value
  3. MIDI CC — sent to the DSP engine (0–127, or 0–16383 for 14-bit CCs)
  4. DSP Float — converted by firmware to physical units (Hz, dB, ms, ratios)

Key insight: Macros control which portion of a CC range your knob covers, and how multiple knobs combine. The physical range of each DSP parameter is fixed in firmware.

Tips & Best Practices
  1. Start from 1:1 Map — Auto-generate a full passthrough, then remove parameters you don't need and lock them to good values.
  2. Name knobs musically — Use "Brightness" instead of "CC 12". Performers should never think about CC numbers.
  3. Use MACRO knobs — Combining multiple CCs into one knob is the most powerful feature. One "Punch" knob beats three separate knobs.
  4. Lock the boring parameters — Use constant mappings for parameters that should stay fixed. Fewer knobs = faster workflow for performers.
  5. Create contrasting presets — Make presets sound dramatically different to showcase the full range of your macro definition.
  6. Use categories — Group presets by style ("Analog", "Digital", "FX") so performers can find sounds quickly.
